-	The app is intended for use for small businesses, small charitable and not-for-profit organisations that may require outreach work to meet and network with customers, sponsors to create bespoke events and analyse the customer interactions, to perform simple cost-benefit analysis reports
-	The stack proposal for the MVP is
o	API integrations of Eventbrite, ZohoCRM and ZohoBooks
o	Authentication via Supabase – works more robustly with the AI layer required for workflow management
o	AI workflow management with Apple Core and Apple Foundation
o	Swift-UI
o	Swift Data models that map directly to the sSupabase tables and external APIs

-	App with 3 user types – public users and admin users with read access and admin users with read and write access
-	Each user has distinct roles and permission levels for app usage
-	Public user has no permission to create, update or delete events – they can browse and search for events that interest them
-	Admin user can create, update and delete events, view user information and AI insights
-	Admin user with read access can only see AI insights for high-level view
